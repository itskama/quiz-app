# Flutter Quiz

Полноценное приложение-викторина (Quiz), написанное на Flutter с использованием паттерна архитектуры **MVVM** (Model-View-ViewModel).

## Особенности
- Архитектура **MVVM** (строгое разделение UI, бизнес-логики и данных).
- State-менеджмент: пакет **provider**.
- Локальная база вопросов в формате `json`.
- 12 вопросов на тему Flutter и Dart.
- Вычисление результата и процентной оценки.

## Требования
- Flutter SDK >= 3.0.0
- Dart SDK >= 3.0.0

## Запуск проекта

1. Склонируйте репозиторий или скачайте исходный код.
2. Перейдите в директорию проекта.
3. Установите зависимости:
   ```bash
   flutter pub get
   ```
4. Запустите приложение:
   ```bash
   flutter run
   ```

## Структура проекта
- `lib/models/` - Модели данных (`quiz_question.dart`, `quiz_result.dart`)
- `lib/data/` - Слой данных (`question_repository.dart`)
- `lib/viewmodels/` - Вьюмодели (`quiz_view_model.dart`)
- `lib/ui/screens/` - Экраны приложения
- `lib/ui/widgets/` - Переиспользуемые UI компоненты
